# Button Creator Web

Aplicação web para criar e personalizar botões CSS em tempo real com visualização do código gerado.

**[Acesse a aplicação aqui](https://kessleru.github.io/ButtonCreator-Web/)**

## Funcionalidades

- Personalização em tempo real do botão
- Controles de texto, cores, tamanho, bordas e fontes
- Geração automática de código CSS
- Persistência local com localStorage

## Tecnologias

- HTML5
- CSS3
- JavaScript Vanilla

## Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador

## Estrutura

```
ButtonCreator-Web/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Controles

- **Texto**: Define o texto do botão
- **Cor**: Cor do texto
- **Cor de fundo**: Background do botão
- **Tamanho/Largura**: Dimensões do botão
- **Borda**: Estilo da borda
- **Raio da borda**: Arredondamento dos cantos
- **Fonte**: Família e tamanho da fonte
